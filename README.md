# 📱 TextTales Interactive

> A chat-based interactive storytelling platform where users experience stories through texting-style conversations.

---

## 🚀 Overview

TextTales Interactive is a full-stack web application that delivers immersive, interactive stories in the form of text message conversations.  

Users progress through narratives by reading and interacting with conversation-style messages, creating a unique and engaging storytelling experience.

---

## ✨ Features

- 💬 **Chat-Based Storytelling**  
  Stories are presented as real-time messaging conversations for immersive interaction.

- 🔐 **User Authentication**  
  Secure login and account management with session-based authentication.

- 🎯 **Freemium Access Model**  
  Users can read a limited number of stories per day, encouraging engagement while controlling usage.

- 🧠 **Dynamic Story Engine**  
  Backend logic processes structured data to deliver story content dynamically.

- ☁️ **Media Integration**  
  Images and media are stored and delivered via Cloudinary.

- 📧 **Email Support**  
  Integrated SendGrid for account-related notifications.
